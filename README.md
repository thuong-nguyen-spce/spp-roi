# spce-roi

## Install

```bash
npm install --save spce-roi
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'spce-roi'
import 'spce-roi/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
